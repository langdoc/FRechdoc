Welcome to FRechdoc pages.

This website contains information about different tools used in Freiburg based language documentation projects.

ELAN

Several of the tools we use are connected to ELAN and the XML files which ELAN uses, as this is one of the most widely used file formats in language documentation work currently. In principle we are always open to explore other alternatives as well. We provide on this page access and documentation to our ELAN templates and annotation conventions.

Data access

Our corpora have been archived to TLA Nijmegen as Pan-Saami and Permic Varieties corpora, but there are also different access mechanisms provided for different users. Please see this and this.

Metadata

Metadata is instrumental to ensure that all resources are easily foundable. We provide access to our CMDI metadata for basic information, but we also acknowledge there are remaining issues with the current model where all metadata is usually stored publicly.
